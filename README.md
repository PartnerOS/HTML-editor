# HTML Content Editor (HTML_editor.py)

Локальный редактор HTML-шаблонов для документов формата **HTML** (например, дипломов/сертификатов) с поддержкой:
- редактирования **жёстко заданных текстов** (без показа HTML-тегов)
- просмотра всех **картинок** в шаблоне (preview, тип, размер)
- **замены изображений** (PNG/JPG) прямо в HTML как `data:image/...;base64,...`

Инструмент появился для решения задач по развитию конструктора партнёрского портала **PartnerOS** (компании LICANT).

---

## Зачем это нужно

В реальных шаблонах для PDF часто есть:
- переменные поля (`{{ ... }}`)
- условные блоки (`{% if ... %}`), например выбор партнёрского статуса и соответствующей плашки

Из‑за этого WYSIWYG‑редактирование обычно не подходит: документ **динамический**, и визуальный редактор не знает, какая ветка шаблона сработает при генерации PDF.

Этот инструмент решает практическую задачу: **безопасно менять контент**, не ломая шаблон.

---

## Возможности

### Вкладка «Тексты»
- список всех найденных текстовых строк (без HTML)
- редактирование выбранного текста
- сохранение обратно в тот же HTML‑файл

### Вкладка «Изображения»
- список всех изображений из шаблона (теги `<img>` и фоновые картинки из CSS)
- preview (фиксированный размер, чтобы интерфейс не “прыгал”)
- замена выбранного изображения на PNG/JPG
- запись изображения в HTML как `data-url` и сохранение файла

---

## Требования

- Python 3.10+ (рекомендуется)
- зависимости:
  - `beautifulsoup4`
  - `Pillow` (для preview и определения размеров картинок)

Установка:

```bash
pip install -r requirements.txt
```

> Без Pillow программа тоже запустится, но preview/размеры могут отображаться не полностью.

---

## Запуск

```bash
python html_editor.py
```

Дальше:
1) Нажми **Открыть HTML**
2) Выбери нужный шаблон (например `diploma.html`)
3) Правь тексты/картинки
4) Сохраняй — изменения пишутся прямо в исходный файл

---

## Рабочие файлы

Временные/рабочие JSON‑файлы сохраняются в папку `_work/` рядом со скриптом (удобно игнорировать в Git).

---

## Формат изображений

Поддерживаются:
- PNG
- JPG/JPEG

При замене изображение сохраняется в HTML как:

```html
src="data:image/png;base64,...."
```

или аналогично для фоновых картинок в CSS (`background` / `background-image`).
